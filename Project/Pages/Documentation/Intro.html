Why another Framework?

With ASP.NET, there is already a very good, scalable solution to develop and host web applications and services based on the .NET framework. But as of 2020, it seems that there are not that many other web development frameworks and servers left besides Microsoft's ASP.NET. This project aims to bring more diversity to the beautiful ecosystem of .NET development, also in the spirit of the open source community.

With the GenHTTP framework, developers should be able to easily develop new web applications in a short time. Everything else is provided by the server infrastructure as well as the excellent ecosystem of .NET Core which easily allows to build, test, and run applications.

Hello World!

To create our first web application using the GenHTTP framework, open a terminal and enter the following command to create a new .NET Core application: 

dotnet new console --framework netcoreapp3.1 -o ExampleWebsite
This will create a new folder ExampleWebsite. Within this folder, run the following command to add a nuget package reference to the GenHTTP Core nuget package and a theme: 
cd ExampleWebsite
dotnet add package GenHTTP.Core
dotnet add package GenHTTP.Modules.Websites
dotnet add package GenHTTP.Themes.Lorahost
You can then edit the generated Program.cs to setup a simple project using the GenHTTP server API: 
using System;

using GenHTTP.Engine;
                        
using GenHTTP.Modules.IO;
using GenHTTP.Modules.Core;
using GenHTTP.Modules.Layouting;
using GenHTTP.Modules.Website;

using GenHTTP.Themes.Lorahost;

namespace ExampleWebsite
{

    public static class Program
    {

        public static int Main(string[] args)
        {
            var project = Layout.Create()
                                .Index(Page.From("Index", "Index Page"))
                                .Add("other", Page.From("Second", "Second Page"));

            var theme = Theme.Create()
                             .Title("Example Website");

            var website = Website.Create()
                                 .Theme(theme)
                                 .Content(project);

            return Host.Create()
                       .Console()
                       .Defaults()
                       .Handler(website)
                       .Run();
        }

    }

}
To run our newly created project, simple execute: 
dotnet run
This will host a new server instance on port 8080, allowing you to view the newly created project in your browser by navigating to http://localhost:8080.
