Hosting with Docker

Currently, there is a bug with the Mono framework that prevents .NET Core 3.0 applications to be deployed via Docker. After the final version has been released, this section will be updated.

Running on Raspberry Pi

To run a .NET Core app on Raspbian OS, you may specify a platform paramter: 

dotnet publish -c Release -r linux-arm
Copy the resulting files from the bin/Release/netcoreapp3.0/linux-arm/publish/ folder onto the target device, mark the application as executable and run it: 
chmod 775 Project
./Project
