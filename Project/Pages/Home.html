Provide Your Own Web Applications

GenHTTP is intended to create simple web applications in a short time using a .NET based development environment.

100% Managed

Written in C# and provided as a .NET Standard 2.1 library, with just a few dependencies.

Open Source

All sources are available on GitHub under the MIT license.

Flexible Hosting

Embed the engine into your existing .NET application or host it via Docker.

Minimal Footprint

Originally intended to run on a netbook with an Intel Atom processor, the server code is optimized to be fast and minimalistic.

Simple & Powerful

The server API provides a simple, yet powerful, fully object-oriented module system to plug in your content.
