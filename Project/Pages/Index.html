100% Managed

Purely written in C# and provided as a .NET Standard 2.0 library, with no further dependencies.

Open Source

All sources are available on GitHub under the MIT license.

Flexible Hosting

Embed the engine into your .NET application or host it via Docker.

Features

Originally intended to run on a netbook with an Intel Atom processor, the server code is optimized to be fast and minimalistic. For example, this website is served by a Raspberry Pi.

The server engine is designed to require no additional dependencies (besides .NET Standard libs). The server process won't fuzz around with your file system, threads, statics, or assemblies.

The server API provides a simple, yet powerful, fully object-oriented module system to plug in your content. The library includes modules for typical concerns such as resource delivery, navigation, templating, or webservices. This allows simple web applications to be written in a short time.

To get started, check out the samples on GitHub.
